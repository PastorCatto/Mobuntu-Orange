# Mobuntu-L4T Build Guide
# Codename: Happy Mask Salesman

## Prerequisites

- Ubuntu 24.04 build host (WSL2 supported)
- zerofree, p7zip-full, git, debootstrap
- Root/sudo access
- Upstream Switchroot L4T scripts (see upstream/UPSTREAM_SOURCES.md)

```bash
sudo apt install zerofree p7zip-full git debootstrap
```

## Setup

1. Clone Mobuntu-L4T and pull upstream:

```bash
git clone <mobuntu-l4t-repo>
cd Mobuntu-L4T
git submodule update --init --recursive
```

2. Download upstream kernel + initramfs per `upstream/UPSTREAM_SOURCES.md`

3. Place kernel/initramfs files per upstream instructions

## Build

```bash
sudo ./build.sh -d switch
```

### Options

| Flag | Description | Default |
|------|-------------|---------|
| `-d` | Device codename | required |
| `-u` | UI override (phosh, plasma-mobile, kde, lxde, mate) | from device.conf |
| `-f` | L4T flavor (kde-noble, gnome-noble, etc.) | from device.conf |

### Examples

```bash
# Default (Phosh, KDE Noble base)
sudo ./build.sh -d switch

# Plasma Mobile
sudo ./build.sh -d switch -u plasma-mobile

# KDE desktop on GNOME Noble base
sudo ./build.sh -d switch -u kde -f gnome-noble
```

## Output

Built image lands in:
```
upstream/l4t-image-buildscripts/output/L4T-<flavor>-image.7z
```

Flash to SD card using Hekate. See `docs/HEKATE_CALIBRATION.md` for
Joy-Con calibration before first boot.

## UI Options

| UI | Type | Notes |
|----|------|-------|
| phosh | Mobile (Wayland) | Primary — recommended for handheld use |
| plasma-mobile | Mobile (Wayland) | KDE mobile experience |
| kde | Desktop (X11) | Full KDE Plasma desktop |
| lxde | Desktop (X11) | Lightweight desktop |
| mate | Desktop (X11) | Traditional desktop |
| ~~gnome~~ | ~~Desktop~~ | **Excluded** — known L4T regressions |

## Notes

- GNOME is explicitly excluded. Do not attempt to add it without verifying upstream L4T compatibility first.
- Joy-Con calibration must be dumped via Hekate before first boot. See `docs/HEKATE_CALIBRATION.md`.
- Kernel and initramfs are NOT generated by this build system. Source from upstream Switchroot.
