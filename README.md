# Mobuntu

A mobile Linux build toolkit for SDM845 devices, originally targeting the Xiaomi Poco F1 (`beryllium`).

---

## Status

**This project is in maintenance/handoff state.** The original maintainers have stepped back from active development.

If you want to continue or fork this project, start with [`HANDOFF.md`](HANDOFF.md). It covers the project history, fork requirements, outstanding issues, and where to start.
